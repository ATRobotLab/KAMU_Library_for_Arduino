# KAMU_Library_for_Arduino
로보까무 제어를 위한 아두이노 라이브러리 입니다.


# How to Use
1. ZIP형식의 소스파일을 다운로드 받습니다.
2. 아두이노 IDE를 실행 한 뒤,  스케치 -> 라이브러리포함 -> .ZIP 라이브러리 추가... 에서 다운로드 받은 KAMU_Library_for_Arduino-main.zip를 추가해 주세요.
3. 라이브러리 설명
   + //kamu.h 라이브러리를 include 합니다.
   + #include "Kamu.h"
